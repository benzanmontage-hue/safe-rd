# SafeRD — Project Overview

## Purpose
Personal safety app for Dominican Republic. One-tap SOS sends GPS location to trusted contacts via WhatsApp, SMS, FCM push, and auto-dials 911.

## Target Audience
- 🇩🇴 Residents concerned about crime
- 🏖️ Tourists visiting DR
- 👩 Women walking alone
- 🏃 Joggers in remote areas
- 🚗 Drivers at night
